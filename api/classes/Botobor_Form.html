      Обёртка для защиты веб-формы

      Основано на http://nedbatchelder.com/text/stopbots.html

      Считается, что форма заполнена роботом, если: - либо между созданием формы и её отправкой прошло слишком мало времени (см. опцию delay); - либо между созданием формы и её отправкой прошло слишком много времени (см. опцию lifetime); - либо заполнено хотя бы одно поле-приманка (см. Botobor_Form::setHoneypot()); - либо заголовок REFERER не совпадает с адресом, где была размещена форма.

      Любая из проверок может быть отключена.

      ИСПОЛЬЗОВАНИЕ

      Файл, создающий форму:

      require 'botobor.php';
…
// Получите разметку формы тем способом, который предусмотрен у вас в проекте, например:
$html = $form->getHTML();
// Создайте объект-обёртку:
$bform = new Botobor_Form($html);
// Получите новую разметку формы
$html = $bform->getCode();

      Файл, обрабатывающий форму:

      require 'botobor.php';
…
if (Botobor_Keeper::get()->isRobot())
{
    // Форма отправлена роботом, выводим сообщение об ошибке.
}
